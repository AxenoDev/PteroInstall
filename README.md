# PteroInstaller

Installation d'installation de Pterodactyl 0.7

## Installation


```bash
apt update
```
```bash
apt full-upgrade
```
```bash
apt install curl wget bash
bash <(curl -s https://raw.githubusercontent.com/AxenoDev/PteroInstall/main/bashall.sh)
```
